Installation Guide:

  1. Install docker ( https://www.docker.com/products/docker-desktop/ )

  2. Download DFT UI images ( https://github.com/storm-dft/dft-ui )

  3. Run browser (Currently only Chrome Supported) ( http://127.0.0.1:8081 )
